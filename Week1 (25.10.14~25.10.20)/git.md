# git
## 기능 구현을 위한 브랜치 생성
```
git checkout -b bloodmoon3929
```

## 기능 구현 후 add, commit
```
git status
git add -A
git commit -m "{원하는 메시지}"
```

## 브랜치 merge (A에 B의 내용을 병합)
```
git checkout A
git merge B
```

## github에 업로드
```
git push origin bloodmoon3929
```

## 과제 제출
브랜치 오른쪽에 있는 "New pull request"
[$미션제목] $이름 미션 제출합니다. 형식으로 Pull request
